The tidyfinance Package

Jun He May 31, 2026

A companion package bundles the small, repetitive tasks that show up in almost every empirical project, so the analysis code can stay focused on ideas rather than plumbing.

One-line data access

The most useful helpers wrap common downloads behind a single function call: Fama–French factors, momentum, macro series, and similar public datasets. Instead of remembering each source's URL and file format, you ask for a dataset by name and a date range and get back a tidy table ready to join.

Consistent conventions

Helper functions enforce conventions — column names, date handling, return units — so that datasets from different sources line up without manual cleanup each time. This consistency is what makes later joins (returns to characteristics, characteristics to factors) painless.

Why wrap it at all

Hiding boilerplate behind named functions has two payoffs: scripts read like the analysis you actually intend, and a fix to a download quirk lives in one place rather than being copy-pasted across many files.
